## ゲーム概要
Unityで製作したファイアーエムブレム風の戦闘マップシステムのひな型を作りました。
これからも様々な機能、そしてストーリーを追加していければと思っています。

## 開発環境
* **Unity**: 6(6000.0.41f1)
* **言語**: C#

## 新しく感じたこと
* ターン遷移をしっかりと間を空けることであったり、メニューを開いている間動作を受け付けないこと、
そしてそのターンの時に該当するユニットがちゃんと動くことこれら至極基本的なことでもそれなに複雑な
管理が必要なんだなということをよく感じました。
* ロジックが必要なゲームジャンルであることから、大変ではありましたがプログラミングの部分ができれば
確実に形になるということが良く感じられ、自身のプログラミングに対する力がついていることがよく感じられた。
