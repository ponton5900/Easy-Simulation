# ターン制シミュレーションの戦闘システム

## ジャンル
- ターン制ストラテジー

### デモ動画
![デモ動画](EasySimulationDemo2.gif)

## ゲーム概要
Unityで製作したファイアーエムブレム風の戦闘マップシステムのひな型を作りました。
これからも様々な機能、そしてストーリーを追加していければと思っています。

## 開発環境
* **Unity**: 6(6000.0.41f1)
* **言語**: C#

## 新しく感じたこと
* ターン遷移をしっかりと間を空けることであったり、メニューを開いている間動作を受け付けないこと、
そしてそのターンの時に該当するユニットがちゃんと動くことこれら至極基本的なことでもそれなりに複雑な
管理が必要なんだなということをよく感じました。
* ロジックが必要なゲームジャンルであることから、大変ではありましたがプログラミングの部分ができれば
確実に形になるということが良く感じられ、自身のプログラミングに対する力がついていることがよく感じられた。

## これからしていきたいこと
* スクリプトのリファクタリング
    1) そのスクリプトが多くの役割を担いすぎないようにすること  
    2) publicである必要のない箇所のチェック  
    3) そのスクリプトに作らなくていい変数を消す  
    4) 座標とユニット両方を保持できる辞書型への変更を検討  
    5) nullチェックを行い、その時の処理を追加する  
* マップのマスの役割をつける（1マスで2つ分の移動力消費など）
* ユニットに勇者、魔王などのスプライトをつける
* ユニットの体力がなくなったときに、リストから消去し、アクセスしないようにする
* ゲーム開始のタイトル画面やゲームオーバー画面、勝利画面等を作って、途中で止まらないようにする
* 絵などは2Dドット絵調にし、AIを活用して絵をつけ、アークナイツのようにストーリーはノベルゲーム風
にし、戦闘マップシステムはこのシステムに様々な機能をつけつつ、使っていきたいと思っています。
